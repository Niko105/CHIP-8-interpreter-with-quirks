# CHIP-8-interpreter-with-quirks
CHIP-8 interpreter capable of supporting the CHIP8, SCHIP, XO-CHIP, and whatnot with some configuration.

Written entirely in Clojure, trying to adhere to the paradigm as much as possible. Uses Java's swing and AWT libraries for graphics with an optional terminal mode without input.
